Test A/B Pixel Connection

After connecting the pixel:

  1. Open browser DevTools (F12)
  2. Go to Console tab
  3. Visit your product page: https://genlabs-dev-store.myshopify.com/products/
  4. Look for these console logs:
[A/B Test Pixel] Initialized
[A/B Test Pixel] Product viewed {productId: "...", variantId: "..."}
[A/B Test Pixel] Fetching test state from https://abtest.dreamshot.io/api/rotation-state
[A/B Test Pixel] Test state result {testId: "...", activeCase: "BASE"}
[A/B Test Pixel] Tracking impression

If you don't see the pixel in Customer Events:

1. First accept the permission update in your app

2. Go to: Your App Settings

3. Look for a permission update notification

4. Accept the new permissions

5. Then return to Customer Events
